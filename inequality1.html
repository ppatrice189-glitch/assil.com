🏆 Extreme Math Olympiad Inequalities 🏆

12 Problems of Extreme Difficulty (in increasing order of challenge)

Problem 1.

Let a, b, c be positive real numbers such that $$a+b+c=3$$. Prove that: $$\frac{a^2}{b} + \frac{b^2}{c} + \frac{c^2}{a} \ge 3$$

Problem 2.

Let a, b, c be non-negative real numbers. Prove that: $$a^3 + b^3 + c^3 + 3abc \ge a^2 b + b^2 c + c^2 a + ab^2 + bc^2 + ca^2$$

Problem 3.

Let a, b, c be positive real numbers. Prove that: $$\frac{a^2}{b} + \frac{b^2}{c} + \frac{c^2}{a} \ge a + b + c$$

Problem 4.

Let x, y, z be positive real numbers such that $xyz=1$. Prove that: $$\frac{1}{x^3(y+z)} + \frac{1}{y^3(z+x)} + \frac{1}{z^3(x+y)} \ge \frac{3}{2}$$

Problem 5.

Let a, b, c be non-negative real numbers. Prove that: $$(a-b)(a-c) + (b-a)(b-c) + (c-a)(c-b) \ge 0$$

Problem 6.

Let a, b, c, d be positive real numbers. Prove that: $$\left(\frac{a}{a+b}\right)^3 + \left(\frac{b}{b+c}\right)^3 + \left(\frac{c}{c+d}\right)^3 + \left(\frac{d}{d+a}\right)^3 \ge \frac{1}{4}$$

Problem 7.

Let a, b, c be positive real numbers such that $$a+b+c=1$$. Prove that: $$\frac{a}{\sqrt{a^2+8bc}} + \frac{b}{\sqrt{b^2+8ca}} + \frac{c}{\sqrt{c^2+8ab}} \ge 1$$

Problem 8.

Let $$a, b, c \ge 0$$. Prove that: $$\sqrt{4a^2+b^2} + \sqrt{4b^2+c^2} + \sqrt{4c^2+a^2} \ge 2(a+b+c)$$

Problem 9.

Let a, b, c be positive real numbers. Prove that: $$\frac{a^2}{b^2+c^2} + \frac{b^2}{c^2+a^2} + \frac{c^2}{a^2+b^2} \ge \frac{3}{2}$$

Problem 10.

Let x, y, z be positive real numbers. Prove that: $$\frac{x^2}{x^2+xy+y^2} + \frac{y^2}{y^2+yz+z^2} + \frac{z^2}{z^2+zx+x^2} \ge 1$$

Problem 11.

Let a, b, c be positive real numbers such that $$abc=1$$. Prove that: $$\frac{1}{a^2+a+1} + \frac{1}{b^2+b+1} + \frac{1}{c^2+c+1} \ge 1$$

Problem 12.

Let a, b, c be positive real numbers. Prove that: $$\sum_{\text{cyc}} \sqrt{\frac{2a}{a^2+b^2}} \le \sum_{\text{cyc}} \frac{a}{\sqrt{ab+bc}}$$
